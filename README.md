# Caesar Cipher

A simple Caesar Cipher program built in Python.

## Features

- Encode messages;
- Decode messages;
- Supports lowercase and uppercase letters;
- Supports digits (0-9);
- Preserves special characters;
- Custom shift value.

## How it works

The program shifts each character in the message by a given number.

- Letters are shifted in the alphabet;
- Digits are shifted in the number list;
- Special characters remain unchanged.

## Project structure

- `main.py` → main program logic
- `ascii_art.py` → logo display
